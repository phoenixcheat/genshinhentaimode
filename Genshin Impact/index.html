For an example, you need to open Lisa in characte

Press numpad 0 to toggle hunting mode, then press F8 to dump textures.

Short guide Download and install 3dmigoto in your Genshin

Once that's done you'll notice 2 Frame Analysis folders in your 3dmigoto folder.

Open the FrameAnalysisDeduped folder.
Open and edit the textures.









Alpha value controls the glowiness so the only thing visible here is the vision.

Go ahead, edit for your needs and then save the file as dds. If you're working with the file from the Deduped folder the filename will tell you which settings to use for saving. In t his case it's BC7 sRGB

Now reload 3dmigoto back in game (F10) and see your changes.
